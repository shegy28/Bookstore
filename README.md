```sh
 -Web Browser
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/shegy28/Bookstore.git
```
### Install

To run on local machine, open your terminal and run the following:

```sh
  npm install
  npm run build 
  npm start
```
